# TeleBox_Plugins_PX

## 简介
个人的 TeleBox 的插件仓库，仅自用，娱乐。

## 技术栈

- **开发语言**: TypeScript
- **数据库**: SQLite (better-sqlite3)
- **任务调度**: node-schedule
- **Telegram API**: telegram (GramJS)
- **图像处理**: Sharp
- **其他依赖**: axios, lodash 等
